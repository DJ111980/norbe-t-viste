# Checklist de produccion

- [ ] D1 production tiene backup reciente.
- [ ] R2 production tiene listado o respaldo de objetos.
- [ ] No hay datos demo.
- [ ] Norbelia puede iniciar sesion.
- [ ] DJ puede iniciar sesion.
- [ ] Admin temporal fue inactivado o rotado.
- [ ] Branding se conserva.
- [ ] Frontend responde.
- [ ] API `/health` responde.
- [ ] CORS permite solo frontend productivo.
- [ ] Tests API pasan.
- [ ] Tests web pasan.
- [ ] Build web pasa.
- [ ] Repo Git esta limpio.

## Antes de dominio propio

- Definir dominio.
- Conectar Pages.
- Configurar route o subdominio API.
- Actualizar CORS.
- Probar SSL, login, imagenes y etiquetas.
