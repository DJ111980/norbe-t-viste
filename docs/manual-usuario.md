# Manual de usuario

## Iniciar sesion

Entrar al frontend de produccion y usar el usuario asignado. Si el sistema pide cambiar
contrasena, hacerlo antes de operar.

## Catalogo

1. Crear categorias.
2. Crear productos.
3. Crear variantes con talla, color y precio de venta.
4. Subir imagenes solo desde la app.

## Lotes e inventario

1. Crear lote de entrada.
2. Agregar detalles con variante, cantidad y costo unitario.
3. Confirmar lote para ingresar stock.
4. Consultar inventario para revisar existencias.

## Ventas

- CONTADO: puede registrar venta sin cliente asociado.
- CREDITO: requiere cliente activo.
- MIXTA: requiere cliente activo y pago inicial.

## Creditos y cartera

Consultar creditos desde el modulo de cartera. Registrar abonos solo cuando el pago sea real.

## Devoluciones

Crear devoluciones desde ventas completadas. Revisar cantidades disponibles antes de confirmar.

## Etiquetas

Imprimir etiquetas por variante o por lote confirmado. El QR identifica la variante.

## Reportes

Usar reportes para ventas, inventario, movimientos, cartera, devoluciones y lotes.
